# 🚀 QC CHECKLIST - 30 SECOND QUICK REFERENCE

## WHERE IS IT?

**Option A (Fastest):**
```
Projects & Campaigns → Click Project → Click Task Card → Click "✅ QC Checklist" Tab
```

**Option B:**
```
Projects & Campaigns → QC Dashboard → Click 👁️ on Task → Click "✅ QC Checklist" Tab
```

## HOW TO USE

1. **Move slider** (0-10) to score each item
2. **Add comment** explaining the score
3. **Click "Save Scores"** to save progress
4. **Check Overall Score** in metrics bar
5. **Click "Approve & Auto-Move"** when ≥78%

---

## KEY FEATURES

✨ **Weighted Scoring**: Important items count more (Weight: 1.0x, 1.5x, 2.0x)
✨ **Auto-Move**: Task moves to next milestone when threshold met
✨ **Gold Standards**: Click 📘 to see benchmarks
✨ **Real-time Calc**: Overall score updates as you change sliders
✨ **Color Coded**: Green=Pass, Orange=Rework, Red=Fail, Gray=Pending

---

## SCORING GUIDE

| Score | Meaning | Action |
|-------|---------|--------|
| 9-10 | Excellent | Pass |
| 8 | Good (Threshold) | Pass |
| 6-7 | Needs Work | Rework |
| 0-5 | Poor | Fail |

---

## EXAMPLE

**Item**: Meta Title & Description  
**Score**: 4/10

**Result**: ❌ FAIL
