# 🎯 HOW TO ACCESS QC CHECKLIST - VISUAL GUIDE

## Method 1: Via Project Kanban View (RECOMMENDED)

```
Main Menu → Projects & Campaigns → Click any Project Card
  ↓
Project Kanban View Opens (6 columns: Ideation, Drafting, QC Review, Design, Publish, SEO Promotion)
  ↓
Click ANY TASK CARD in any column
  ↓
Task Details Dialog Opens with title: "🧱 Task Details & QC Workflow"
  ↓
Look at the TOP of the dialog - you'll see TWO TABS:
  Tab 1: "Task Details"    ← (default, basic info)
  Tab 2: "✅ QC Checklist" ← **CLICK THIS TAB**
  ↓
✅ QC CHECKLIST NOW VISIBLE!
```

### What You Should See:
1. **Metrics Bar** at the top:
   - Items Scored: 7/10
   - Average Score: 7.2
   - Pass Threshold: 78%
   - Overall Weighted Score: 82%
   - QC Result: Pass

2. **Two Accordion Sections**:
   - Content QC (6 items)
   - SEO QC (4 items)

3. **Each Item Shows**:
   - Item name with info icon (ℹ️)
   - Score slider (0-10)
   - Weighted score progress bar
   - Comment textarea
   - "View Gold Standard" button

## Method 2: Via QC Dashboard

```
Main Menu → Projects & Campaigns
  ↓
Click the "QC Dashboard" tab (3rd tab at top)
  ↓
QC Tracker & Training section appears
  ↓
Look at the TABLE - you'll see columns: Task Name, Project, Milestone, etc.
  ↓
Find any task row and click the 👁️ (Eye icon) in the Actions column
  ↓
"QC Task Review" Dialog Opens
  ↓
Look at the TOP - you'll see TWO TABS:
  Tab 1: "QC Review"           ← (default, basic QC form)
  Tab 2: "✅ QC Checklist"    ← **CLICK THIS TAB**
  ↓
✅ QC CHECKLIST NOW VISIBLE!
```

## 🔍 What If You Don't See It?

### Troubleshooting Checklist:

✅ **Step 1**: Confirm you're in the right place
- Are you in "Projects & Campaigns" section?
- Did you click on a project card or go to QC Dashboard?

✅ **Step 2**: Look for the tabs
- Do you see a dialog/modal opened?
- Look at the VERY TOP of the dialog
- You should see tab buttons

✅ **Step 3**: Click the correct tab
- First tab is usually "Task Details" or "QC Review"
- **Second tab** should say "✅ QC Checklist"
- Click that second tab!

✅ **Step 4**: Check dialog size
- Dialog should be large (takes up most of screen)
- If too small, try zooming out your browser (Ctrl + -)

## 📸 Key Visual Indicators

Look for these elements to confirm you're seeing the QC Checklist:

1. ✅ **Title**: "🎯 QC Evaluation & Scoring"
2. ✅ **Metrics Bar**: Shows "Items Scored", "Average Score", etc.
3. ✅ **Accordion Sections**: Expandable sections like "Content QC" or "SEO QC"
4. ✅ **Sliders**: Each item has a horizontal slider (0-10)
5. ✅ **Buttons**: "View Gold Standards", "Save Scores", "Approve & Auto-Move"
6. ✅ **Progress Bars**: Weighted score shown as progress bars
7. ✅ **Color Coding**: 
   - Green = Pass
   - Orange = Rework
   - Red = Fail
   - Gray = Pending

## 🎬 Quick Start Actions

Once you see the QC Checklist:

### For First-Time Users:
1. Click "📘 View Gold Standards" to see benchmarks
2. Expand "Content QC" accordion to see items
3. Try moving a slider to see weighted score update
4. Add a comment in the textarea
5. Click "Save Scores" to save your work

### For QC Reviewers:
1. Expand accordion sections one by one
2. Score each item using the slider
3. Add specific feedback in comments
4. Reference gold standards as needed
5. Check overall weighted score in metrics
6. Click "Approve & Auto-Move" when ready

### For Content Creators:
1. Review items marked as "Rework" or "Fail"
2. Read QC comments for specific issues
3. Check gold standards to understand requirements
4. Make necessary improvements
5. Resubmit for review

## 💬 Common Questions

**Q: How many items are in the QC Checklist?**
A: 10 items total - 6 for Content QC, 4 for SEO QC

**Q: Can I save partially completed checklists?**
A: Yes! Click "Save Scores" at any time. Items with score 0 show as "Pending"

**Q: What happens when I click "Approve & Auto-Move"?**
A: The task automatically moves to the next milestone in the workflow

**Q: Why is the button disabled?**
A: Overall Weighted Score must be ≥ Pass Threshold (usually 78%)

**Q: Can I see past QC scores?**
A: Yes, click the "History" button (if available) or check the task history

**Q: What's the difference between Weight and Score?**
A: Score is 0-10 rating, Weight is multiplier (1.0x, 1.5x, 2.0x). More important items have higher weight.
